# Advanced Media Analytics

A free, open-source textbook introducing Communication students to R for measuring and analyzing digital media data. Written by **Jenny Jiang & Brian Walsh**.

Read the book at: **https://bwalsh5.github.io/advanced-media-analytics/**

## About

This book teaches students in Communications programs (Journalism, Public Relations, Strategic Communications, and related fields) how to use R and the tidyverse to analyze digital media data — without requiring an extensive statistics background.

Topics covered:

- Data manipulation and visualization with the tidyverse
- Basic and inferential statistics
- Statistical analysis of one, two, and multiple variables
- Text analysis (sentiment analysis, word frequency, topic modeling)
- Network analysis of social media connections
- Geospatial analysis and data mapping
- Introduction to Python and AI tools
- Publishing data analysis projects
